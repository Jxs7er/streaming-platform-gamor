# Gamor Proyect
This proyect is split in two parts:
- **/client** (frontend)
- **/server** (backend)

---

# Requirements: 
Make sure you've installed:
- Node.js (>= 18)
- npm

---

## 1. Install dependencies
### .1. Client
```bash
cd client
npm install
```

### .2. Server
```bash
cd server
npm install
```
## 2. Running the project
### .1. Client
```bash
cd client
npm run dev
```
or
```bash
npm run dev -- --host
```
### .1. Server
```bash
cd server
npm run dev
```
